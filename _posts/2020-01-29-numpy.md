## Numpy



- Numpy 모듈 임포트

```python
import numpy as np
```



- 배열 생성하기

```python
arr1 = np.array([1, 2, 3])
arr2 = np.array([[1, 2, 3], [4, 5, 6]])
```



- type(), .dtype, .astype, .shape, .ndim

```python
type(arr1)
> class 'numpy.ndarray'

arr1.dtype
> int 64

arr1.astype('int32')
arr1.dtype
> int 32

arr1.shape
> (3,)
arr2.shape
> (2,3)

arr1.ndim
> 1
arr2.ndim
> 2
```
